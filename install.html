Install Guide¶

Install Yender¶

Type below commands:

$ git clone https://github.com/okdshin/Yender.git --recursive
$ pip install ./Yender

Yender depends on these Python packages:

  • NumPy

All these libraries are automatically installed with Yender if needed.
